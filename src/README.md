# Frontend Components for AI-Native Textbook

This directory contains the React components for the AI-Native Textbook project.

## Components Overview

### Personalization
- `PersonalizationButton.js`: A component that allows users to set their preferences (skill level, background, learning preferences) to personalize their learning experience.

### Translation
- `TranslationToggle.js`: A component that allows users to toggle between English and Urdu translations of the textbook content.

### AI Q&A
- `QABot.js`: An interactive component that allows users to ask questions about the textbook content and receive AI-generated answers.

### Exercises
- `Exercises.js`: Interactive exercises and quizzes to test the user's understanding of the content.

### Progress Tracking
- `ProgressTracker.js`: A component that tracks and displays user progress through chapters and modules.

### Curriculum Navigation
- `Curriculum.js`: A component that visualizes the learning path and allows navigation through the curriculum.

## Usage

These components are designed to be integrated into the Docusaurus-based documentation system. Each component is self-contained and manages its own state.
