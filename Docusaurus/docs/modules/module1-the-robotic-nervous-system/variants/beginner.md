# ROS 2 کا تعارف - بینر ویرینٹ

## ROS 2 کیا ہے؟

تصور کریں کہ ایک روبوٹ انسان کی طرح کام کر رہا ہے. اس کے دماغ کے ذریعے، اس کے ہاتھ، پاؤں، آنکھیں، اور دیگر اعضا مواصلت کرتے ہیں. ROS 2 اسی طرح کام کرتا ہے، جیسے ایک روبوٹ کا "نروس سسٹم"، جس سے اس کے مختلف حصے ایک دوسرے کے ساتھ بات چیت کر سکتے ہیں.

## بنیادی اجزاء

### 1. نوڈز (Nodes)
- ایک روبوٹ کے اندر ایک "عمل" یا "کام".
- مثال: ایک کیمرہ نوڈ تصاویر لیتا ہے، اور ایک موتور نوڈ چلنے کا کم کرتا ہے.

### 2. ٹاپکس (Topics)
- نوڈز کے درمیان پیغامات کا تبادلہ کرنے کا ذریعہ.
- مثال: ایک کیمرہ ایک "ٹاپک" پر تصویریں بھیجتا ہے، اور کئی دیگر نوڈز اس "ٹاپک" سے تصاویر حاصل کر سکتے ہیں.

### 3. خدمات (Services)
- ایک درخواست دینے اور جواب حاصل کرنے کا طریقہ.
- مثال: ایک سروس سے کہا جا سکتا ہے "2 + 2 کتنے ہوتے ہیں؟" اور یہ فوری طور پر "4" جواب دے گا.

## عملی وضاحت

### پبلشر اور سبسکرائبر

1. **پبلشر**: تصاویر بھیجنے والا نوڈ.
2. **سبسکرائبر**: تصاویر حاصل کرنے والا نوڈ.

ان کا تعلق ایک دوسرے سے ہے مگر وہ ایک دوسرے کے بارے میں کچھ نہیں جانتے. صرف "ٹاپک" کا نام معلوم ہوتا ہے.

## سادہ عملی مثال

**سادہ پبلشر**:
```python
# ایک نوڈ کے طور پر 'ہیلو' پیغام بھیجنا
print('Hello Robot!')
```

**سادہ سبسکرائبر**:
```python
# ایک نوڈ کے طور پر 'ہیلو' پیغام حاصل کرنا
message = input('کیا میسج ملا؟ ')
print(f'میں نے یہ حاصل کیا: {message}')
```

## مشق

اپنے کمپیوٹر پر "ہیلو ورلڈ" پروگرام لکھیں اور سمجھیں کہ یہ ایک سادہ "پبلشر" ہے.

یہ ویرینٹ بینر لیول کے لوگوں کے لیے ہے تاکہ وہ ROS 2 کے بنیادی تصورات کو سمجھ سکیں.
